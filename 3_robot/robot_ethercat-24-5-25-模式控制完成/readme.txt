
备注：
1.网口2初始化时间300ms；网口1插网线初始化时间2.3s，不插网线初始化时间3.36s。网口1的延时等待检测网线是否插上时间不能变更短，若更短会导致网线检测错误。
2.下发配置参数前需等待超过4s时间，不然会导致主cpu错误，可以在主cpu加读主卡状态指令，主卡配置完成即上报状态。
3.HCMD_WRITE_IOTBL 耗时 2.4s。
4.HCMD_WRITE_CONFIG 耗时2.2s 主要是ECAT从站模块进入OP状态耗时2s。
5.正常IO输入输出时间为210-230us之间，该时间为每一个背板通讯耗时。
